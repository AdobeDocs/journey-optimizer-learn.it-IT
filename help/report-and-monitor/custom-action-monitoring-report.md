# Monitorare le azioni personalizzate

Scopri come monitorare in modo efficace lo stato operativo e le prestazioni delle chiamate HTTP effettuate dai tuoi percorsi agli endpoint di terze parti tramite azioni personalizzate. Ottieni visibilità sugli errori HTTP, sui colli di bottiglia della latenza e sui problemi di limite che possono influenzare la consegna.

Questo report aiuta a identificare i problemi, migliorare le prestazioni del percorso e garantire un funzionamento senza problemi.
