# Monitorare le azioni personalizzate

Scopri come monitorare in modo efficace lo stato operativo e le prestazioni delle chiamate HTTP effettuate dai tuoi percorsi verso gli endpoint di terze parti tramite le azioni personalizzate.Ottieni visibilità sugli errori HTTP, sugli ostacoli relativi alla latenza e sui problemi di limitazione che potrebbero influire sulla consegna.

Questo rapporto aiuta a identificare i problemi, migliorare le prestazioni del percorso e garantire un funzionamento senza problemi.
