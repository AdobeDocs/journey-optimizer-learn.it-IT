# Creare le raccolte

Scopri come creare raccolte per la gestione delle decisioni. Le raccolte vengono utilizzate per gestire le offerte in gruppi logici e sono necessarie per generare le attività di gestione delle decisioni. Le raccolte dinamiche consentono di raggruppare le offerte in base ai tag, mentre le raccolte statiche possono essere curate manualmente.
