# Attiva percorso all’invio del modulo

Questo caso d’uso dimostra come l’invio di un modulo per il cliente attivi un percorso Adobe Journey Optimizer in tempo reale, abilitando comunicazioni di follow-up immediate e personalizzate in base alle informazioni fornite dall’utente.


## Prerequisiti per questa esercitazione

* Accesso ad Experience Platform

* Nozioni di base sui concetti di Experience Platform (profili, pubblico, set di dati)

* Familiarità con Journey Optimizer

## Passaggi di implementazione

* Definire uno schema evento in Adobe Experience Platform
* Configurare un evento personalizzato in Adobe Journey Optimizer
* Configurare il canale e-mail
* Creare un Percorso attivato dall’evento personalizzato
* Inviare dati di invio modulo a Adobe Experience Platform
