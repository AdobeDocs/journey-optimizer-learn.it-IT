# Utilizzare il targeting di più entità con pubblico federato nei percorsi

Scopri come utilizzare il targeting di più entità per andare oltre il pubblico basato su profili e fornire esperienze cliente più rilevanti utilizzando un pubblico federato.

Questo video illustra come eseguire il targeting dei percorsi in base a ID iscrizione, ID account o ID ordine, per un’attivazione più precisa del pubblico e un coinvolgimento personalizzato quando un singolo cliente è associato a più entità business.

Per ulteriori informazioni su questa funzione, consulta [la documentazione](https://experienceleague.adobe.com/it/docs/journey-optimizer/using/orchestrate-journeys/manage-journey/supplemental-identifier){target="_blank"}.
