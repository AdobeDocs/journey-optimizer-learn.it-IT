# Esportare i set di dati in percorsi di archiviazione cloud

Scopri come esportare un set di dati da Journey Optimizer a una destinazione Amazon S3.

>[!AVAILABILITY]
>La funzione di esportazione dei set di dati è in versione beta e disponibile per tutti gli utenti di Adobe Journey Optimizer. Per accedere alle destinazioni, contatta il tuo rappresentante Adobe.

>[!NOTE]
>Durante la fase beta, tutti i set di dati sono visibili. Una volta che questa funzione è disponibile, sarà limitata ai soli set di dati generati dal sistema.

>[!VIDEO](https://video.tv.adobe.com/v/3418688/?quality=12&learn=on)

Per ulteriori informazioni, consulta la [documentazione del prodotto](https://experienceleague.adobe.com/docs/journey-optimizer/using/data-management/datasets/export-datasets.html?lang=en).
